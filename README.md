## 简介
仅本地化 Mod。界面保持简体中文，但人物名、家族名、地名、头衔、文化、宗教/信仰、政体等**显示为英文**。
中日韩（CJK）专有名词保留原样：中国、日本、朝鲜地区的人名/地名/王朝名仍为汉字/假名/谚文（例：李世民、日本、高丽、唐/宋/元/明）。蒙古名按英文显示（成吉思汗 → Genghis Khan）。

## 已知问题
- **绰号**无法处理：绰号键在所有文化间共用，改英文会破坏 CJK 例外，故一律保留中文。
- 少数运行时生成的头衔在个别情况下可能仍显示中文。
- 与其他改动相同 simp_chinese 键的本地化 Mod 冲突——请将本 Mod 排在加载顺序**最后**。
- 针对 1.19.0.6 制作；游戏更新新增的键在 Mod 更新前不会被覆盖。

## 使用方法
启动器启用本 Mod，游戏语言设为**简体中文**，加载顺序放最后。
